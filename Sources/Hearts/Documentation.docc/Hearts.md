# ``Hearts``

A command-line tool for generating emoji-art from images.

## Overview

Hearts converts images into emoji-art that can be displayed in terminals, chat applications, or anywhere that renders emoji in a fixed-width font.

```bash
swift run Hearts -w 80 image.png
```

The tool supports local files and URLs, various emoji selection options, and outputs directly to stdout.
